Let the user easily change simple functions in WooCommerce that can affect sales. 

== Description ==

Let's the user enable "Direct checkout", which basically skips the shopping cart page.
Customizable add to cart buttons. Custom text and CSS code. On both single product pages and all other WooCommerce pages.
Customizable "Buy now" button.
Checkout page functions: Let the user add some images below payment form.
Let the user add in conversion tracking pixel code.

== Installation ==

1. Login to your WordPress
2. Go to Plugins -> Add New
3. Click Upload Link
4. Browse to the woosale-converter.zip file and choose that file
5. Click the "Install Now" button
6. Wait while plugin is uploading to your server
7. Click "Activate Plugin" button
8. Now you can create your first Popup
That's all!

Usage

All buy buttons can be customized with CSS.
We recommend to use http://css3buttongenerator.com/ , but of course this is just an option. Any CSS code will do.
